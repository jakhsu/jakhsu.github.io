介紹如何用hexo在github page上部屬靜態網站

前言

身為軟體工程師，很重要的一件事是把自己所學紀錄整理下來。不僅能夠省下未來重複查找文件、StackOverflow的時間，還能幫助自己內化所學，或是與他人分享討論。這時候部落格，特別是靜態網站，就是很方便快速的選項。

這次想分享的工具有

  • Hexo，由Tommy Chen開發，非常熱門的static site generator。
  • github pages，是由github提供的hosting服務，可以免費託管個人或是專案網站

設定

hexo提供的設定都放在根目錄 _config.yml 檔案裡，這邊介紹一些基本的設定

  • Site
    • title: 網站名稱
    • subtitle: 副標
    • description: 網站描述
    • keywords: 可供搜尋引擎查找的關鍵字，似乎有利於SEO，但是這還需再研究
  • Extentsions
    • theme: 網站layout主題
  • Deployment
    • deploy: 部屬相關設定

這邊特別說明一下theme、deploy這兩個比較重要的設定。
